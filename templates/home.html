Step 1 : Select Font Style or Upload customized Font Style (.ttf/.otf file)

Step 2 : Upload Template(.jpg/.png/.jpeg) and Position Text by clicking anywhere and rearraging them by drag and drop

Step 3 : Upload CSV File containing details in same order (i.e. from top to bottom and left to right)

Step 4 : Get the Zip of Certificates

Features provided by CertiGini

  • Customized Certificates.
  • Dynamically Text Positioning along with increase and decrease text size buttons.
  • Drag and Drop Text Elements anywhere on the certificate.
  • Uploading Customized Font Files.
  • Generate a bundle of certificates(in the form of a zip) by providing csv file with all the details.
